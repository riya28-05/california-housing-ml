# California Housing Price Prediction

## Overview
This project compares Linear Regression and Decision Tree models to predict housing prices using the California Housing dataset.

## Dataset
- 20,640 records
- 8 features
- Target: Median House Value

## Models Implemented
- Linear Regression
- Decision Tree Regressor

## Results
| Model | RÂ² Score | MSE |
|-------|----------|------|
| Linear Regression | 0.57 | 0.55 |
| Decision Tree | 0.62 | 0.49 |

## Key Insights
- Median Income is the most important feature (~53% importance)
- Decision Tree captures non-linear patterns better than Linear Regression
